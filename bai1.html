HTML Basic Examples

1. HTML Basic

All HTML documents must start with a document type declaration: !DOCTYPE html.
The HTML document itself begins with html and ends with /html.
The visible part of the HTML document is between html and /html. 

            <!DOCTYPE html>
            <html>
            <head>
                <title>Title</title>
            </head>
            <body>
                My First Heading
                My First Paragraph
            </body>
            </html>

2. HTML Elements

The HTML element is everything from the start tag to the end tag:

                <tagname>content</tagname>

                <p>This is a paragraph</p>

3. HTML Attributes

HTML elements can have attributes.
Attributes provide additional information about elements.
Attributes are always specified in the start tag.
Attributes usually come in name/value pairs like: name="value".

                <tagname attribute="value">content</tagname>

                <a href="https://www.w3schools.com">This is a link</a>

4. HTML Headings

HTML headings are defined with the <h1> to <h6> tags.

                <h1>This is heading 1</h1>

                <h2>This is heading 2</h2>

                <h3>This is heading 3</h3>

                <h4>This is heading 4</h4>

                <h5>This is heading 5</h5>

                <h6>This is heading 6</h6>

5. HTML Paragraphs

HTML paragraphs are defined with the <p> tag.

                <p>This is a paragraph</p>

6. HTML Styles

You can add styles to HTML elements.
The most common way to add styles is to use an HTML style attribute.
The style attribute can contain any CSS property.

                <tagname style="property:value;">content</tagname>

                <h1 style="color:blue;">This is a blue heading</h1>

7. HTML Formatting

HTML also defines special elements for defining text with a special meaning.

                <b>This text is bold</b>

                <strong>This text is strong</strong>

                <i>This text is italic</i>

                <em>This text is emphasized</em>

                <mark>This text is marked</mark>

                <small>This text is small</small>

                <del>This text is deleted</del>

                <ins>This text is inserted</ins>

                <sub>This text is subscripted</sub>

                <sup>This text is superscripted</sup>

8. HTML Quotations

HTML blockquotes are defined with the <blockquote> tag.

                <blockquote>This is a blockquote</blockquote>

9. HTML Comments

HTML comments are not displayed in the browser, but they can help document your HTML source code.

                <!-- This is a comment -->

10. HTML Colors

HTML colors are specified with predefined color names, or with RGB, HEX, HSL, RGBA, or HSLA values.

                <h1 style="color:blue;">This is a blue heading</h1>

                <p style="color:red;">This is a red paragraph</p>

11. HTML CSS

CSS stands for Cascading Style Sheets.
CSS describes how HTML elements are to be displayed on screen, paper, or in other media.
CSS saves a lot of work. It can control the layout of multiple web pages all at once.
External stylesheets are stored in CSS files.

                <link rel="stylesheet" type="text/css" href="styles.css">
